# Какой ключ используется с командой `ls` для отображения всех файлов, включая скрытые?
- [x] `-a`
- [ ] `-l`
- [ ] `-h`
- [ ] `-r`

## :desc
Ключ `-a`, сокращение от `all`, отображает все файлы, включая скрытые (начинающиеся с точки).
